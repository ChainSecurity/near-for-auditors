# NEAR for Advanced Users

## How to Use:

### Manual Installation:

* Create a new python virtual env
* ``pip install mkdocs``
* ``pip install mkdocs-material``
* ``mkdocs serve``

### Docker:

In project's root directory run:

``docker run --rm -it -p 8000:8000 -v ${PWD}:/docs squidfunk/mkdocs-material:8.1.9``
